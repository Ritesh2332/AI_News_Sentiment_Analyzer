# AI News Sentiment Analyzer
🔗 **Live Demo:** https://newspipe.streamlit.app/ 

An end-to-end **machine learning-based web application** that analyzes real-time news sentiment using NLP techniques. The app fetches live news via RSS feeds and web scraping, classifies sentiment (positive, neutral, negative), and presents insights through an interactive Streamlit dashboard.

---

## 🚀 Features

* 📰 Real-time news collection using RSS feeds & web scraping
* 🤖 Machine learning-based sentiment analysis using NLP
* 📊 Sentiment classification: Positive, Neutral, Negative
* 🗂️ Category-based filtering (Technology, Business, Sports, etc.)
* 🌐 Interactive Streamlit web interface
* ⚡ Fast and lightweight processing

---

## 🧰 Tech Stack

* **Python** – Core programming
* **Streamlit** – Web application framework
* **BeautifulSoup** – Web scraping
* **Requests** – API & HTTP handling
* **Pandas** – Data processing
* **TextBlob** – NLP & sentiment analysis

## ▶️ Run Locally

### 1. Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/AI-News-Sentiment-Analyzer.git
cd AI-News-Sentiment-Analyzer
```

### 2. Create Virtual Environment

```bash
python -m venv venv
venv\Scripts\activate
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the App

```bash
streamlit run app.py
```

---

## 📊 How It Works

1. Fetches real-time news using RSS feeds and web scraping
2. Cleans and processes text data
3. Applies NLP-based machine learning model for sentiment analysis
4. Displays results in an interactive Streamlit dashboard
